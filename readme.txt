Overview

The Document Chatbot is a desktop application built with PyQt5 that allows users to interact with documents and ask questions about their content. The chatbot utilizes the OpenAI GPT-4 API to provide answers based on the extracted text from various document formats, including PDF, DOC/DOCX, and TXT files.
Features

Load Document: Load and extract text from PDF, DOC/DOCX, or TXT files.
Ask Questions: Interact with the chatbot by asking questions about the content of the loaded document.
Save Output: Save the chat history to a text file.
